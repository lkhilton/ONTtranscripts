# ONTtranscripts

## Requirements

Conda: `https://docs.conda.io/en/latest/miniconda.html`

Snakemake: `conda install -c bioconda snakemake`

## Data

Download and unzip the tarballs containing fast5 and fastq files to the data directory
```data
├── fast5
│   ├── IX7311
│   └── IX7312
├── fastq
└── metadata```
